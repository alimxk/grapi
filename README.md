# GrAPI review
GrAPI is a lightweight HTTP service that serves a wide collection of configurable JSON objects that can be used to test your UI.


### Usage

```bash

# Install grapi universally

npm -i -g grapi

# Start
grapi -p 3030

# Usage

Http://localhost:3030/users
```
